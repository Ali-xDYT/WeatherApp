# WeatherApp

Simple weather application created in JS using [Weather-JS](https://www.npmjs.com/package/weather-js)
## Installation

```sh
git clone https://github.com/Ali-xDYT/WeatherApp.git
cd WeatherApp
npm install
```

## Usage

```sh
npm start
```
